# Портфолио Backend разработчика


## 🚀 Возможности

- **Портфолио**: Современный одностраничный сайт с разделами:
  - О себе
  - Навыки и технологии
  - Проекты
  - Контакты
  

- **Темная/Светлая тема**: Для портфолио и админ-панели





## 🛠 Технологии

- React
- TypeScript
- Tailwind CSS
- shadcn/ui компоненты
- Lucide React (иконки)

## 📝 Структура проекта

```
├── components/
│   ├── admin/          # Компоненты админ-панели
│   ├── portfolio/      # Компоненты портфолио
│   └── ui/            # UI компоненты (shadcn)
├── contexts/          # React контексты
│   ├── AdminContext.tsx
│   └── ThemeContext.tsx
└── styles/           # Глобальные стили
```
